# Data-structure-Algorithm
Data structure&amp;Algorithm in C++

Happy New Year~

用简单的写法整理下我理解的数据结构和基础算法。
在每个完成的cpp文件下都附有相应的简单测试。
为求代码简洁，少数数据结构或者算法会直接使用STL container作为底层容器而不是自己从头实现(必要的话还是会重写的)。
